PLAKAR-TOKEN(1) - General Commands Manual

# NAME

**plakar-token** - Manage Plakar tokens

# SYNOPSIS

**plakar&nbsp;token**
\[**create**]

# DESCRIPTION

The
**plakar token**
command manages tokens used to authenticate to Plakar services.
Set the
`PLAKAR_TOKEN`
environment variable to use a token, and see
plakar-login(1)
to persist it in the configuration.

# SUBCOMMANDS

**create**

> Create a new token.

# SEE ALSO

plakar(1),
plakar-login(1)
